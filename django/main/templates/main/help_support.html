Help & Support

Find answers to common questions about accounts, clubs, events, messaging, notifications, and executive features on HoosLinked.

Getting Started

What is HoosLinked?

HoosLinked is a web application where UVA students can explore clubs, join organizations, choose roles, and manage or attend events.

Who can use HoosLinked?

HoosLinked is designed for all UVA students.

How do I create an account?

Click the profile or login icon in the top-right corner, then select Sign Up. You can create an account using your information or sign up with Google.

Can I use HoosLinked without logging in?

You can browse clubs and events, but you won’t be able to join clubs or RSVP to events.

How do I update my profile?

Click the profile icon in the top-right corner, then select Edit Profile.

Account Help

How do I delete my account?

Click the profile icon in the top-right corner, go to Settings, then select Delete Account.

Why is my profile picture not showing?

If your profile picture is not updating, try refreshing the page or uploading the image again.

What happens if I sign up using Google?

After signing up with Google, you will be taken to your profile page where you can edit your information.

Clubs

How do I browse clubs?

Click the Clubs option in the top navigation bar to view all available organizations.

How do I search for a specific club?

Go to the Clubs page and use the search bar labeled Search for clubs.

How do I save a club?

Open a club page and click the Save button. Saved clubs will appear on your profile page.

How do I become a member of a club?

Go to the club’s page and click Become Member.

Where can I find club announcements?

Open a club page and scroll down to view announcements, discussions, events, and contact information.

Events

How do I find events?

Click the Events tab in the navigation bar to view upcoming events sorted by date.

How do I RSVP to an event?

Open an event and click the RSVP button.

How do I see who is attending an event?

On the event page, check the Attendees section to see who is going.

How do I add an event to my calendar?

Open the event and click the calendar option. This will redirect you to Google Calendar where you can add the event.

Messaging

What is the Messages button for?

The Messages feature allows you to chat with other users, including club members and executives.

How do I start a message?

You must be logged in. Click the Messages button, then search by name, email, or username.

Why does messaging ask for an email or name?

You can search for users using their email, name, or username. This helps locate their account.

Does entering an email send a real email?

No. It only helps find the user in the system and does not send an actual email.

Who can I message?

You can message any user, including club members and executives.

Executive Users

How do I become a club executive?

You need an executive code provided by the system administrators. Eligible executive users may receive this code by email.

What is an executive code?

An executive code is required to gain executive access for a specific club.

Where do I enter the executive code?

Go to the club page, enter the code in the executive code field, then click Verify Executive.

What can executives do?

Executives can create events, post announcements, manage members, and require approval for users who want to join.

How do executives manage members?

Executives can approve, remove, or ban members and control whether users must request approval to join.

How do executives create announcements or events?

Go to the Executive Dashboard, where you will find options to create announcements and events.
